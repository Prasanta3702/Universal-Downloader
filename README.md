# UniversalDownloader

A lightweight desktop application for downloading videos and audio from supported platforms with an easy-to-use interface.

## Features

- Download video and audio content
- Multiple quality options
- Progress tracking
- Simple desktop UI
- Fast and responsive downloads
- Cross-platform Java support

## Installation

### Download Installer

Download the latest version from the Releases section.

Latest release:
https://github.com/YOUR_USERNAME/UniversalDownloader/releases

### Run the Installer

1. Download the `.exe`
2. Run the installer
3. Follow the setup instructions
4. Launch UniversalDownloader

## Usage

1. Copy a supported video URL
2. Paste it into UniversalDownloader
3. Select desired quality
4. Choose download location
5. Click Download

## Requirements

- Windows 10 / Windows 11
- Internet connection
